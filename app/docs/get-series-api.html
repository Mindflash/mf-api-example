Use this endpoint to get series information. By omitting the :seriesId parameter, a full list of all series in the organization is returned. By supplying the :seriesId parameter, more specific data to a series is returned.

Name Type Optional Value
:seriesId URL YES The Mindflash ID of the series to search for.

Examples

Example: Get a list of all series in organization 
GET https://api.mindflash.com/api/v2/series
Response: 200
