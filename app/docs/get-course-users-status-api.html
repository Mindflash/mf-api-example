Use this endpoint to get a list of all users enrolled in a specific course. Some extra information will be returned including the status in the course and basic completion information if applicable.

Name Type Optional Value
:courseId URL NO The Mindflash ID of the course the to query for user information.

Examples

Example: Get a list of users enrolled in a specific course. 
GET https://api.mindflash.com/api/v2/course/1662438386/user
Response: 200
