Use this endpoint to add multiple users to a group.

Name Type Optional Value
:groupId URL NO The Mindflash ID of the group to add the users to. 
{
  "userIds": []
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.
Field Optional Notes
userIds NO An array of Mindflash user IDs to add to the group.

Examples

Example: Add 2 users to a group. 
POST https://api.mindflash.com/api/v2/group/1662730985/user

{
  "userIds" : [1234, 6789]
}

Response: 200
