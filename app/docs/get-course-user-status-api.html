Use this endpoint to get detailed course enrollment information for a specific user in a specific course.

Name Type Optional Value
:courseId URL NO The Mindflash ID of the course the user is enrolled in.
:userId URL NO The Mindflash ID of the user to get information for.

Examples

Example: Get a users course enrollment information. 
GET https://api.mindflash.com/api/v2/course/1662438386/user/1662427377
