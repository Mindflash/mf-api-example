Use this endpoint to get a list of all users enrolled in a specific series. Some extra information will be returned including the status in the series and basic completion information if applicable.

Name Type Optional Value
:seriesId URL NO The Mindflash ID of the course the to query for user information.
status Querystring YES Filter results by series status. Can be one of:
Completed
Invited
Passed
Failed
Started

Examples

Example: Get a list of users enrolled in a specific series. 
GET https://api.mindflash.com/api/v2/series/1662438388/user
Response: 200
