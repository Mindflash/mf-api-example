Use this endpoint to get a list of courses in a specific series. By omitting the :courseId parameter, a full list of all courses in the series is returned. By supplying the :courseId parameter, more specific data to the course is returned.

Name Type Optional Value
:seriesId URL NO The Mindflash ID of the series to get a list of courses for.
:courseId URL YES The Mindflash ID of the course in the series to get information for. The course must already be part of the series.

Examples

Example: Get all courses in a series. 
GET https://api.mindflash.com/api/v2/series/1662438388/course
Response: 200

Example: Get a specific course in a series. 
GET https://api.mindflash.com/api/v2/series/1662438388/course/1662438387
Response: 200
