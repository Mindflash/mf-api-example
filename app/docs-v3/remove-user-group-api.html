Use this endpoint to remove a single user from a group.

Name Type Optional Value
:groupId URL NO The Mindflash ID of the group to remove the user from.
:userId URL NO The Mindflash ID of the user to remove from the group.

Examples

Example: Remove a user from a group. 
DELETE https://api.mindflash.com/api/v3/group/1662730985/user/1662730983
Response: 200
