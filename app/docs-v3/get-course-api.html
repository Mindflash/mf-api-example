Use this endpoint to get course information. By omitting the :courseId parameter, a full list of all courses in the organization is returned. By supplying the :courseId parameter, more specific data to a course is returned.

Name Type Optional Value
:courseId URL YES The Mindflash ID of the course to search for.
status Querystring YES Filter results by course status. Can be one of:
Active
Archive
All (Default)

Examples

Example: Get a list of all courses in organization 
GET https://api.mindflash.com/api/v3/course
