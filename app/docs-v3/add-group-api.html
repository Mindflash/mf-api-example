Use this endpoint to add user group information.

Name Type Optional Value
responseFormat Query String YES Format the response. Possible values are:
array
json
 
{
  "groupName": "{a group name}"
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.

Examples

Example: Add User Group to an organization 
POST https://api.mindflash.com/api/v3/group

{
  "groupName" : "a group name"
}

Response: 200
