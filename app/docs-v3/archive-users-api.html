Use this endpoint to archive multiple users. Archived users will be unable to take courses or login to the Mindflash site. 

{
  "userIds": []
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.
Field Optional Notes
userIds NO An array of Mindflash user IDs to archive.

Examples

Example: Archive 3 users. 
POST https://api.mindflash.com/api/v3/user/archive

{
  "userIds": [4567, 7898, 9988]
}
					
Response: 200
