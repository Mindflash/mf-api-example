Use this endpoint to get detailed course enrollment information for a specific user in a specific series.

Name Type Optional Value
:seriesId URL NO The Mindflash ID of the series the user is enrolled in.
:userId URL NO The Mindflash ID of the user to get information for.

Examples

Example: Get a users series enrollment information. 
GET https://api.mindflash.com/api/v2/series/1662438388/user/1662427377
