Use this endpoint to add users to the Mindflash system. The user information supplied must contain the basic information, but can optionally include group information and course enrollment information.
Note: A maximum of 200 users may be added per api call. 

{
  "users": [
    {
      "firstName": "",
      "lastName": "",
      "email": ""
    }
  ],
  "requiredCourseIds": [],
  "courseIds": [],
  "groupIds": [],
  "clientDatestamp": "",
  "batchId" : 1
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.
Field Optional Notes
users NO A list of all users information to be added to the Mindflash system.
Field Optional Max Length Notes
firstName NO 35 Users first name.
lastName NO 35 Users last name.
email YES* 254 Users email address.
username YES* 254 Unique username for user.
permissions YES N/A One of:
administrator
manager
trainer
trainee (Default)
yammerId YES N/A Yammer ID of user.
password YES 35 A login password. Passwords allow the user to login via the Mindflash website.
* Note: At least one of email or username must be supplied. Both may be supplied.
requiredCourseIds YES A list of Mindflash course IDs to enroll the users in. All users will be enrolled in these courses with the required option enabled.
courseIds YES A list of Mindflash course IDs to enroll the users in. All users will be enrolled in these courses with the required option disabled.
groupIds YES A list of Mindflash group IDs to place the users in. All users will be placed into these groups.
clientDatestamp NO The current date. All dates must be in the format YYYY-MM-DD.
batchId YES This must be a unique number (preferably sequential) you must generate to identify this specific api call. Upon success, this number will be returned in the response.

Examples

Example: Add 2 trainees and enroll in 3 courses, 1 which is required. 
POST https://api.mindflash.com/api/v3/user

{
  "users": [
    {
      "firstName": "Tom",
      "lastName": "Jones",
      "email": "tjones@yourorg.com"
    },
    {
      "firstName": "Rumple",
      "lastName": "Stilskin",
      "email": "rumple@yourorg.com"
    }
  ],
  "requiredCourseIds": [ 1234 ],
  "courseIds": [ 6789, 6543 ],
  "groupIds": [],
  "clientDatestamp": "2013-07-17",
  "batchId" : 122
}
					
Response: 200
{
  "batchId": 122
}
