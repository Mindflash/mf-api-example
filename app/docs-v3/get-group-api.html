Use this endpoint to get user group information. By omitting the :groupId parameter, a full list of all groups in the organization is returned. By supplying the :groupId parameter, all user records in that group are returned.

Name Type Optional Value
:groupId URL YES The Mindflash ID of the group to search for. If the value 0 is supplied, the result will be all ungrouped users in the organization.

Examples

Example: Get a list of all groups in organization 
GET https://api.mindflash.com/api/v3/group
Response: 200
[
  {
    "id": 1662730984,
    "name": "Test Group 1",
    "userCount": 2
  },
  {
    "id": 1662730985,
    "name": "Test Group 2",
    "userCount": 1
  }
]

GET https://api.mindflash.com/api/v3/group/1662730985
