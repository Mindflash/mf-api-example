Use this endpoint to edit a user's basic information.

Name Type Optional Value
:userId URL NO The Mindflash ID of the user to edit. 
{
  "firstName": "",
  "lastName": "",
  "email": "",
  "username": "",
  "password" : ""
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.
Field Optional Max Length Notes
firstName YES 35 Users first name.
lastName YES 35 Users last name.
email YES 254 Users email address.
username YES 254 Unique username for user.
password YES 35 A login password. Passwords allow the user to login via the Mindflash website.

Examples

Example: Edit a user, change username and password. 
POST https://api.mindflash.com/api/v3/user/333

{
  "email": "tjones.new@yourorg.com",
  "password" : "newpass"
}
					
Response: 200
