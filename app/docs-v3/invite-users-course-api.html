Use this endpoint to invite multiple users to a specific course.

Name Type Optional Value
:courseId URL NO The Mindflash ID of the course to invite the users to. 
{
  "userIds": [],
  "clientDatestamp": "2013-07-17",
  "required": false
}
Note When posting json data in the body of the request, the Content-Type must be set to application/json.
Field Optional Notes
userIds NO An array of Mindflash user IDs to invite to the course.
clientDatestamp YES The current date. All dates must be in the format YYYY-MM-DD.
required YES Set true to make the course required for the users. Default is false.

Examples

Example: Invite 3 users to a course. 
POST https://api.mindflash.com/api/v3/course/1662438386/invite

{
  "userIds": [4567, 7898, 9988],
  "clientDatestamp": "2013-07-17",
  "required" : false
}
					
Response: 200
