Use this endpoint to add a single user to a group.

Name Type Optional Value
:groupId URL NO The Mindflash ID of the group to add the user to.
:userId URL NO The Mindflash ID of the user to add to the group.

Examples

Example: Add a user to a group. 
POST https://api.mindflash.com/api/v2/group/1662730985/user/1662730983
Response: 200
